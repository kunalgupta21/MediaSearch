# 🚀 Media Search App

A modern and premium React media search application that allows users to instantly search and explore **Photos, Videos, and GIFs** with a stunning UI, smooth animations, and responsive experience.

Built with **React + Redux Toolkit + Tailwind CSS + Vite**.

---

# ✨ Features

* 🔍 Search Photos, Videos & GIFs
* ⚡ Fast and responsive UI
* 🎨 Modern glassmorphism design
* 🌈 Smooth animations and hover effects
* 📱 Fully responsive layout
* 🖼 Dynamic media grid
* 🎬 Video previews
* ✨ GIF support
* 🧠 Redux state management
* 🚀 Optimized with Vite
* 🌙 Premium dark theme interface

---

# 🛠 Tech Stack

* React
* Redux Toolkit
* Tailwind CSS
* Vite
* Unsplash API

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/kunalgupta21/media-search-app.git
```

Move into the project folder:

```bash
cd media-search-app
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
VITE_UNSPLASH_KEY=your_unsplash_api_key
VITE_PEXELS_KEY=your_pexels_api_key
VITE_GIF_KEY=your_gif_api_key
```

---

# 🚀 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
